## Instalacion

1) Clonar el Repositorio:
  
  ``` bash
  git clone https://github.com/AlonsoErnesto/Facebook_clone.git
  1.1) FRONTEND
  cd frontend 
  npm i
  modificar archivo .env (para conectar backend)
  npm start
  1.2) BACKEND
  cd backend 
  npm i
  modificar .env (cloudinary, mongobd, email,etc)
  npm run dev
  
  ```
  
2) Listo!
